    当 DOM 树结构发生变化时（例如元素尺寸、位置的变化），浏览器需要重新计算所有元素的位置和几何属性，这个过程称为回流。它是昂贵的操作，因为它可能涉及到整个文档树的重新布局。
    回流后通常伴随着重绘，即重新绘制受影响的区域。如果只是样式变化（如颜色改变），则只会触发重绘而不会引起回流，但这仍然是一个耗资源的过程。
