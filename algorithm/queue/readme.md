- cpu缓存 
举个例子 从1000-1019 cpu访问下标0时就可以将1000-1019的连续内存加载到缓存中

连续内存空间：由于数组元素连续存储，当访问一个元素时，其相邻的元素很可能已经被加载到缓存中。这提高了缓存命中率，减少了缓存未命中的开销。
链表：由于链表元素在内存中的位置不连续，访问链表元素时可能会导致频繁的缓存未命中，降低了缓存利用率。
